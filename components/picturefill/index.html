Picture Fill - Responsive Images

Resize this page to see the image source change.

How to use

  1. Attach the script js/plugins/picture-fill.min.js
  2. Create a picture element in the header: 
    
<script>document.createElement( "picture" );</script>
  3. Add your image and define breakpoints for each version (see source) 
    
<img alt="" 
src="//placehold.it/640x320/0099C7/ffffff&text=Small" 
sizes="(min-width: 40em) 100vw, 100vw" 
srcset="//placehold.it/640x320/0099C7/ffffff&text=Small 540w, 
//placehold.it/768x384/E8117F/ffffff&text=Medium 768w, 
//placehold.it/840x420/27B987/ffffff&text=Large 840w">
