How to use

  1. Attach the dependency script js/vendors/classie.js
  2. Attach the script js/plugins/scroll-events.js
  3. Plugin appends class="scrolled" to the body when you start scrolling so you can trigger events like pinning the menu to the top.
  4. Looks for the nav with the ID main-nav and adds class="active" to each nav item when page is scrolled to that section.
