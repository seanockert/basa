How to use

  1. Attach the script js/plugins/social-buttons.js
  2. Use the button style in scss/plugins/_social-buttons.scss. This includes the images.
  3. Change the target URL in the script to your site (currently stackoverflow.com). This grabs number of tweets and shares for that URL.
  4. Add the buttons: <a href="..." class="social twitter/facebook"><span>Tweet/Share</span><b></b></a>
