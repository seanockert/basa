Lightbox

A simple lightbox with no dependencies.

How to use

Javascript: lightbox.js

<script src="js/plugins/lightbox.js"></script>

<!-- Add rel="lightbox" to the anchor link to open the full image in a lightbox: -->
<a href="http://placehold.it/640x280" rel="lightbox">
  <img src="http://placehold.it/128x128">
</a>
